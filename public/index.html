Tower of Funtasy

The best Tower of Fantasy Discord bot

Tower of Funtasy is a fan-made Discord bot written with Discord.js that provides several useful information about the game Tower of Fantasy.It will be the perfect tool for ToF community servers.

We Provide The Best

Global and CN data

The bot supports the global version and the NC version of the game, which includes the difference between simulacra, relics...

ToF order simulator

The bot has a order simulator system which allows to simulate all the types of orders of Tower of Fantasy, gold nucleus, black nucleus ...

Daily informations

The bot changes this information according to the day we are. There is the possibility to change its own time zone.
